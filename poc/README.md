# 하불대사 3원 대사 엔진 — PoC

협력사 하불(지급운임)을 **3원(三元) 대조**해 과오/미달/근거없는 청구를 자동 적발하는
**돌아가는 데모**. 하불대사의 가장 어려운 핵심(정답금액 산출 + 대사)을 IT·백엔드 없이 먼저 검증.

## 무엇을 증명하나
- ① **안전운임** = 앱의 **실제 고시 데이터(2022-07) + 계산 로직(calcMulti/applySur) 그대로 재사용** → 법적 하한
- ② **당사 내역**(세방 인정: 운임+합의 추가비용) — PoC라 샘플
- ③ **협력사 명세서**(청구액) — PoC라 샘플
- → 항목·총액 3원 대조 → **협력사 과다/과소청구 · 안전운임 미달(법 위반 위험) · 근거없는 추가청구** 자동 플래그

## 실행
```bash
node poc/habul_daesa_poc.js
```

## 단계
- **1단계 — 대사**: 3원 대조로 이상 적발 + 대사표 출력
- **2단계 — 통보 메일 초안 자동 생성**(발송 X, 승인 후 발송 전제)
  - 메일 A(배차담당자·내부): 이상 건 전체 요약
  - 메일 B(협력사·외부 ★초안): **협력사가 과다·근거없이 청구한 건만**
    (안전운임 미달은 당사 내부 시정 대상 → 협력사 통보 제외 ← 도메인 규칙 반영)

## 시연 시나리오(샘플 4건)
1. 정상(모두 일치) → 이상 없음
2. 협력사 **대기료 과다청구**(3만 vs 5만) → 차액 2만 적발 → 협력사 통보 O
3. 당사 운임이 **안전운임 미달**(협력사는 정당 청구) → 당사 시정, 협력사 통보 X
4. **근거없는 추가청구**(셔틀비, 당사 내역에 없음) → 적발 → 협력사 통보 O
